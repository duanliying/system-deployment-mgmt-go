# Docs @ Web Client APP for Service Deployment Agent & Service Deployment Agent Manager

### How to run Web Client APP

 - pip install Flask
 - pip install requests
 - pip install PyYAML
 - python run.py

 * Now you can use your own Web Client App on <your IP>:<5555>
